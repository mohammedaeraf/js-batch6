## ⭐ Bonus Problem (Optional)

### Salary Calculator using Functions

Create a function called `calculateSalary` that calculates the **net salary of an employee**.

The function should accept the following parameters:

* `basicSalary`
* `hra` (House Rent Allowance)
* `da` (Dearness Allowance)
* `tax`

The function should calculate:

**Gross Salary**

```
grossSalary = basicSalary + hra + da
```

**Net Salary**

```
netSalary = grossSalary - tax
```

The function should **return the Net Salary**.

---

### Example

```javascript
Input:
basicSalary = 30000
hra = 5000
da = 2000
tax = 3000

Output:
Net Salary = 34000
```

---

### Requirements

* Create a function named `calculateSalary`
* Accept **four parameters**
* Calculate **Gross Salary**
* Calculate **Net Salary**
* Return the **Net Salary**
* Print the result using `console.log()`

---

### Expected Sample Output

```javascript
Net Salary = 34000
```
