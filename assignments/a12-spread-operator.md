# 📄 Assignment 12: Spread Operator in JavaScript

## 🎯 Objective

Practice using the **Spread Operator (`...`)** with arrays, objects, and functions.

---

## 🧩 Part 1: Arrays

### 🔹 Task 1: Copy an Array

Create an array:

```javascript
const colors = ["Red", "Blue", "Green"];
```

👉 Create a copy using spread operator and print both arrays.

---

### 🔹 Task 2: Merge Arrays

```javascript
const fruits1 = ["Apple", "Banana"];
const fruits2 = ["Orange", "Grapes"];
```

👉 Merge both arrays into a new array and print the result.

---

### 🔹 Task 3: Add Elements

```javascript
const numbers = [2, 3, 4];
```

👉 Create a new array:

- Add `1` at the beginning
- Add `5` at the end

👉 Print the updated array.

---

## 🧩 Part 2: Objects

### 🔹 Task 4: Copy Object

```javascript
const student = {
  name: "Rahul",
  age: 15,
};
```

👉 Create a copy of this object and print it.

---

### 🔹 Task 5: Update Object

Using the same `student` object:

👉 Create a new object:

- Change age to `16`
- Add a new property: `grade: "10th"`

👉 Print the updated object.

---

### 🔹 Task 6: Merge Objects

```javascript
const address = {
  city: "Bhatkal",
  state: "Karnataka",
};
```

👉 Merge `student` and `address` into a new object and print it.

---

## 🧩 Part 3: Functions

### 🔹 Task 7: Spread in Function Call

```javascript
const nums = [10, 20, 30];
```

👉 Create a function `add(a, b, c)` that returns sum.

👉 Call the function using spread operator and print result.

---

## 🧩 Part 4: Challenge Task ⭐

### 🔹 Task 8: Real-Life Example

Create an object:

```javascript
const cart = {
  item: "Shoes",
  price: 2000,
};
```

👉 Create a new object:

- Add `quantity: 2`
- Update price to `1800`

👉 Print final object.
